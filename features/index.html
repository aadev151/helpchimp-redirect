Функции веб-сайта HelpUnicorn.ru

  • Регистрация пользователей (благотворительных организаций и исполнителей);
  • Подтверждение электронной почты;
  • Выкладывание, редактирование и удаление задания пользователями;
  • Возможность просмотра заданий неавторизованными пользователями (чтобы фонды не теряли потенциальных исполнителей, не желающих регистрироваться);
  • Фильтрация заданий по популярности и новизне, а также по тегам и фондам;
  • Возможность сброса пароля;
  • Возможность поиска фондом подходящего исполнителя (топ исполнителей и статистика выполнения);
  • Личный чат между исполнителями и фондами (для сохранения анонимности личных контактов) на сайте; связь неавторизованных пользователей с фондами через email;
  • Работа с API для разработчиков сторонних сервисов;
  • Электронная волонтерская книжка "Книга Добрых Дел";
  • Конструктор сайтов для благотворительных организаций;
  • Карточки пользователей (статистика выполнения / просмотр выложенных фондом заданий) - короткий адрес uncorn.me/username. Примеры исполнителя и фонда.
